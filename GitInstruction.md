## Ветки. Локальная работа с ветками в git.
Для поиска текущих веток нужно использовать команду git branch
Чтобы создать новую ветку используйте команду git branh 'название ветки'
Если есть необходимость удалить ветку, которую вы не слили с другой, то используйте команду git branch -D 'название ветки'
Если же нужно удалить ветку после слияния, то команду git branch -d 'название ветки'
И для самого слияния веток, перейдите в основную ветку, а после используйте команду git merge 'название ветки'
